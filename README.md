# 響け！ユーフォニアム

## Project Setup:

```bash
# clone this repository
git clone https://github.com/kochan4php/hibike-euphonium.git

# go to project directory
cd hibike-euphonium

# install dependencies
yarn install #or npm install

# run the development server
yarn dev #or npm run dev
```
